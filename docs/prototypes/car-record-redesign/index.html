跨端统一的车辆保养工具设计稿

这个 HTML 是可点击原型，不是营销页。它以业务 PRD 为依据，把提醒、记录、车辆管理、备份恢复和调试日期收敛到同一套移动端组件系统，适合后续分别落到 iOS 与 Android。

导航模型提醒、记录、我的三 Tab；新增记录为全局浮动动作。
组件体系统一卡片、标签、表单、底部弹层、分段控件和反馈 toast。
业务约束同车同日唯一、当前应用车辆、日期格式和项目稳定 ID 均体现在交互中。
跨平台落地不依赖 iOS 专属控件，Android 可复用相同信息结构与视觉 token。
点击底部导航可切换三大模块。
点击右下角加号可新增保养记录并更新里程。
点击车辆卡片可切换当前应用车辆。
